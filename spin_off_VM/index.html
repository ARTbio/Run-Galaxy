Spin off a virtual Machine

  1. Go to the Google Cloud Dashboard and select "Compute Engine" on the left hand menu bar
  2. Select the submenu "Instances de VM"
  3. Click on the top bar menu the "CREER UNE INSTANCE" panel
  4. Put name "bare-galaxy"
  5. Choose a Zone (suggestion: europe-west1-c)
  6. Type de machine: choose 2 vCPU with 7.5 Memory
  7. Disque de Démarrage: Click on Modifier
  8. Select Ubuntu 14.04 LTS
  9. At the bottom of the form, put 100 Go for the Disk Size (Taille)
  10. Leave the selection Disque persistant standard / Standard persistant drive
  11. Click Select / Sélectionner
  12. Click Authorize HTTP traffic / Autoriser le traffic HTTP
  13. Click Créer / Create

Connect to the started virtual Machine

After a few seconds, the VM turns on "green" and an ssh menu becomes selectable

  1. Roll down this ssh menu and select the first option Ouvrir dans la fenêtre du navigateur
  2. A shell console pop out and you should now be ready to control your VM with linux command lines
  3. Try to enter the drosofff@bare-galaxy:~$ sudo -i command and hit the return key.
  4. The unix prompt become root@bare-galaxy:~#, you are mastering your VM as root administrator !
  5. [Optional] Here, if you do not have to work with the VM, you can turn off the VM and even trash it:
  6. in one shot, go back to your VM control panel in the web browser, ensure that the running VM is checked, and press the Trash button in the top menu.
  7. Confirm that you want to trash the VM and loose everything.
  8. after a few seconds the VM disappears from the Dashboard.
